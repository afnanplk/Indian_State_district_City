## ⚡ Quick Start

### Download

```bash
git clone https://github.com/afnanplk/Indian_State_district_City.git
```

### Node.js

```javascript
const locations = require("./india_locations.json");

console.log(locations[0].state);
```

### JavaScript

```javascript
fetch("india_locations.json")
  .then(res => res.json())
  .then(data => console.log(data));
```

### Python

```python
import json

with open("india_locations.json", encoding="utf-8") as f:
    locations = json.load(f)
```

## ⚙️ Dataset Generator

The repository also includes a Node.js generator:

```
fetch_cities.js
```

### What it does

- Downloads raw location datasets
- Extracts States, Districts and Villages
- Flattens nested structures
- Removes duplicates
- Sorts locations alphabetically
- Generates the final optimized JSON

### Run

```bash
node fetch_cities.js
```
